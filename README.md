# WiFi-Booster
Script ini work di usb-WiFi ALFA AWUSO36H, apa salahnya mencoba untuk jenis WiFi merk lain.
Eksekusi:

cd ~/Downloads

chmod 755 WiFi-Booster.sh
 
./WiFi-Booster.sh

Note:
Untuk setting Region bisa kalian ubah sendiri, info lebih lanjut:
